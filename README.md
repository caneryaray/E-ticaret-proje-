Proje Açık
laması:
Bu proje, evcil hayvan sahipleri için köpek ve kedi elbiseleri satmayı amaçlayan bir e-ticaret sitesidir. Proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiştir ve temel web geliştirme becerilerini göstermektedir.

Özellikler:

Ana Sayfa (index.html): Ürünler görsellerle listelenir ve kullanıcı dostu tasarım sunar.

Ürün Sayfaları: Köpek ve kedi ürünleri ayrı kategorilerde gösterilir.

Login Sistemi: Kullanıcı giriş ve çıkış işlemleri için basit bir login sayfası içerir.

JavaScript Etkileşimleri: Sepete ürün ekleme, filtreleme ve basit dinamik özellikler.

Responsive Tasarım: Mobil ve desktop uyumlu tasarım.

Kullanılan Teknolojiler:
HTML, CSS, JavaScript

Proje Amacı:
Temel web geliştirme bilgilerini kullanarak, gerçek dünyaya uygun bir e-ticaret deneyimi oluşturmak ve frontend tasarım ile interaktif kullanıcı deneyimi becerilerini geliştirmek.
